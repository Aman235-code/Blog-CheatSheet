  1. HTML Boilerplate: The Boilerplate is nothing but starting template that is used to write html code 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
   Only this statement will be displayed in the browser output 
</body>
</html>
  2. p Tag: The <p> tag defines the paragraph tag.It is used when we are writing any paragraph, 
    <p>This is a paragraph,a normal paragraph</p>
  3. Comments:The Comments in HTML are used to document the code,so that it is easy to understand what that piece of code is doing, it is written as, 
    <!--This statement will not be displayed into the output-->
  4. The Heading TagThe heading tag describes the heading of your page.There are normally 6 headings from <h1> to <h6>.The highest font size is heading 1 and the lowest font-size heading is heading 6 
    
                    <h1>I'm the largest heading</h1>
                    <h2>I'm the second largest heading</h2>
                    <h3>I'm the third largest heading</h3>
                    <h4>I'm the fourth largest heading</h4>
                    <h5>I'm the fifth largest heading</h5>
                    <h6>I'm the least heading</h6>
  5. The anchor tag The anchor tag is used to deal with links of any website. It is used to navigate from one web page to other web page,the url of the webpage that you want to visit is given in the href attribute.If you want to go to the google's web page 
    <a href="www.google.com">Click me to go to the google's webpage</a>
  6. The image tag The <img> tag is used to add an image to the html page. It also has an alt attribute that is used as an alternate text to that image when the image is not loaded.The name or the complete path to that image is given in the src attribute. 
    <img src="home.jpg" alt="This is a home image">
  7. Bold,Italic and Underline tags The bold tag is used to make the text bold, and the italic tag is used to make the text as italic and if you want any text to be underlined underline tag is used. 
    
                    <b>I'm Bold boy</b>
                    <i>I'm italic boy</i>
                    <u>I'm an underlined text</u>
  8. New Line and Horizontal line tagsThe <br> tag is used to add a new line between any two sentences or the paragraphs. The <hr> tag is used to add a horizontal line that acts as a separator between any two HTML elements. 
    
                    <p>I'm line number 1</p>
                    <br>
                    <p>I'm line number 2</p>

                    <p>Below me there is a horizontal line</p>
                    <hr>
                    <p>Above me there is a horizontal line</p>
  9. Sub-Script and SuperScript tags The subscript tag just lowers the character,and superscript tag just raises the character. 
    
                    The chemical formual of the water is:
                    H<sub>2</sub>0

                    The polynomial equation is:
                    ax<sup>3</sup>+bx<sup>2</sup>+cx+d
  10. Preserving the white spaces HTML always ignores the extra spaces. But if you the extra spaces included in the html page add pre tag. 
    <pre> you are away from me                        Yes I'm away from you</pre>
  11. HTML Lists The Lists are used to store the elements sequentially(one after the other) So,html lists are also used for the same. They are enclosed within the <li> </li> tags. Ordered lists:They are numbered and enclosed withing the <ol> </ol> tags. 
     
                    <ol>
                        <li>Bread</li>
                        <li>Butter</li>
                        <li>eggs</li>
                        <li>soap</li>
                    </ol>
  12. HTML unordered List They are not numbered just they are written in the bullet points they are written inside the <ul> </ul> tags. 
    
                <ul>
                    <li>Bread</li>
                    <li>Butter</li>
                    <li>eggs</li>
                    <li>soap</li>
                </ul>
  13. HTML Table Tag: Just like a normal table,HTML table also used to design a table to store the values.It has the rows and the columns.
